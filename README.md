# AI Text Summarizer

An AI-powered web app built with Streamlit and Groq API that summarizes any text instantly.

## Tech Used
- Python 3.13
- Streamlit
- Groq API (LLaMA 3.3 model)
- python-dotenv

## How to Run

### 1. Clone the repo
git clone https://github.com/Valarithmetic/ai-summarizer.git
cd ai-summarizer

### 2. Install packages
pip install streamlit groq python-dotenv

### 3. Add your API key
Create a .env file:
GROQ_API_KEY=your-groq-key-here

### 4. Run the app
streamlit run app.py
